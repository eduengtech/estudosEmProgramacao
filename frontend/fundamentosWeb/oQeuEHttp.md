## O que é HTTP?

**HTTP** é um protocolo de camada de aplicação usado para transferir dados em uma rede. Ele é responsável por definir as regras da comunicação
entre cliente e servidor, ou seja, HTTP define como os dados devem ser enviados e recebidos na Web.

Assim como um restaurante possui regras para realizar pedidos,
o protocolo HTTP define como o cliente faz a requisição e como o
servidor responde.
